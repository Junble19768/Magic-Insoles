# Project Brief（项目基石）

> 所有其他记忆文件的源头。定义「做什么、边界在哪」。
> 最近更新：2026-07-06

## 一句话定位

面向儿童（4–12 岁）的智能步态矫正鞋垫 AIoT 系统——参赛作品（嵌入式芯片与系统设计竞赛，主题「AI 赋能设计，设计点亮 AI！」）。

## 核心目标

1. **实时可视化**：BLE 接收 32 路压力值 → Three.js 足底热力图 + COP 轨迹
2. **边缘推理**：STM32 TinyML 步态分类（正常/内八/外八）+ 置信度显示 + LED/马达异常反馈
3. **云端 AI**：STM32 经 LTE 直传原始帧 → 后端存储 → DeepSeek 生成每日健康日报
4. **远程查看**：手机 Web 拉取历史数据与 AI 报告（PC 端 Dashboard 远程监控）

## 核心价值主张

- 低成本硬件（FSR）+ AI 语义增强 = 类专业步态分析能力
- 边缘-云协同 AI 分层：STM32 端 TinyML 实时推理 + 云端 LLM 生成报告
- 完整 AIoT 闭环：采集 → 传输 → 分析 → 反馈

## 范围边界

### 本人负责

- **嵌入式侧**：数据协议设计、FSR 数据处理流程、BLE/LTE 通信逻辑、TinyML 输入输出接口
- **后端**：Python FastAPI 服务、数据接入存储、DeepSeek LLM 调用、报告生成
- **前端**：手机 Web（React + Three.js）+ PC 远程监控后台，BLE 实时热力图、COP 轨迹、AI 日报展示

### 硬件组负责

- PCB 设计打板、传感器底层驱动、TinyML 模型训练部署、LED/马达控制等纯硬件/固件层面工作

## 硬件现状（已打板完成）

| 硬件 | 规格 | 说明 |
|------|------|------|
| 主控 | STM32 自研 PCB | 已设计打板 |
| 压力传感器 | FSR 鞋垫，32 路（左脚 16 / 右脚 16） | 当前为分区峰值，非连续分布 |
| 物联网模组 | LTE + GPS + BLE 一体模组 | 抽象 AT 指令层 |
| 姿态传感器 | IMU（三轴加速度 + 三轴角速度） | 辅助步态事件检测 |
| 反馈器件 | LED 指示灯 + 震动马达 | 步态异常时实时提醒 |
| 显示 | OLED 小屏幕 | 低优先级 |

**FSR 数据说明**：空间分辨率有限，不做物理压力场重建，而是通过深度学习进行**步态语义特征提取**（足底压力的高维语义表示）——核心技术亮点表述。

## 成功标准

- 比赛视频演示闭环：BLE 热力图实时可见 + AI 日报可展示
- 人手有限，**功能从简，保证核心演示闭环**
- TinyML 模型只需自行采集少量数据 + 简单过拟合，够演示即可
