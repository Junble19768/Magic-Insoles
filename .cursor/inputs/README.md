# Inputs（人工输入区）

本目录**不允许 AI 修改**，只由人工维护。它是引入外部设计意图、需求与讨论记录的入口，供 AI 在开发时阅读、引用，但不会被 AI 写入或改写。

- `ai-discusses/`：用户与其他 AI（如 ChatGPT/Gemini）的交流记录，用于沉淀设计理念、工作哲学等对 AI 开发关键但难以从代码推断的背景（如 [`TALK-0001.md`](ai-discusses/TALK-0001.md)）。
- `demands/`：外部需求（甲方 / PRD / 需求文档等），当前为占位目录。

仓库整体的需求背景、目录结构与 Rules/Skills 说明见 [`.cursor/README.md`](../README.md)。
